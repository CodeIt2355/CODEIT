Coding Exercises

Exercise 1: HTML Basics

Write the html code to create a simple webpage with the following elements:

  1. A heading with the text "Welcome to CODE IT"
  2. An unordered list with three items: "html", "CSS", "JavaScript"
  3. An image with the source "logo.jpg" and alt text "CODE IT Logo"

Exercise 2: CSS Styling

Add CSS styling to the html code from Exercise 1 to achieve the following:

  1. Change the background color of the webpage to light blue
  2. Center align the heading text
  3. Apply a border to the image with a thickness of 2 pixels and a solid black color
